# SpringResume
My personal resume - written in Latex

# Linux
- Clone the repo

- Install relevant packages
sudo pacman -S texlive-binextra         # for hyperref, bookmark
sudo pacman -S texlive-latexextra       # for fontawesome5, titlesec, geometry
sudo pacman -S texlive-fontsextra       # for sourcesanspro
sudo pacman -S texlive-science          # for amsmath
sudo pacman -S texlive-pictures         # for xcolor, graphics

- run `pdflatex resume.tex` in the repository in which the resume exists


# Windows
